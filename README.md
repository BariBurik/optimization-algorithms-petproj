# Методы оптимизации: Gradient Descent, Momentum, Newton

## 📌 Описание проекта

В данном проекте реализованы и сравнены три базовых метода оптимизации:

- Градиентный спуск (Gradient Descent)
- Метод Momentum
- Метод Ньютона (Newton Method)

Цель проекта — исследовать поведение методов при минимизации нелинейной функции, а также сравнить их сходимость и устойчивость при разных начальных условиях.

---

## 📐 Исследуемая функция

Рассматривается функция двух переменных:

f(x, y) = x² + y² − 2xy²

### Градиент функции:

∇f(x, y) = [
    2x − 2y²,
    2y − 4xy
]

### Гессиан:

H(x, y) = [
    [2,   −4y],
    [−4y, 2 − 4x]
]

---

## ⚠️ Особенность функции

Функция **не имеет глобального минимума**, так как не ограничена снизу.

Однако существует **локальный минимум в точке (0, 0)**, относительно которого проводится сравнение методов.

---

## ⚙️ Реализованные методы

### 1. Gradient Descent

Классический метод:

w_{k+1} = w_k − η ∇f(w_k)

---

### 2. Momentum

Ускоренный градиентный метод:

v_{k+1} = α v_k + η ∇f(w_k)  
w_{k+1} = w_k − v_{k+1}

---

### 3. Newton Method

Использует вторые производные:

w_{k+1} = w_k − η H^{-1}(w_k) ∇f(w_k)

В реализации добавлена стабилизация гессиана:

H_stable = H + εI

---

## 🧪 Методика эксперимента

1. Генерируется **5 случайных стартовых точек**
2. Для каждой точки запускаются все 3 метода
3. Для каждой точки вычисляется **оценка качества**:

score = расстояние до (0,0) + 0.3 * |f(w)|

4. Выбирается **лучшая стартовая точка**
5. Все методы сравниваются **из этой одной точки**
